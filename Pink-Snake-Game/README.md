# Snake Game 🐍

A browser-based interactive Snake game built with **HTML, CSS, and JavaScript**. Use the arrow keys to control the snake, eat food to score points, and avoid hitting walls or yourself. Simple, fun, and fully playable in the browser!

## Features

- 20x20 grid game board
- Arrow key controls for the snake
- Score counter displayed on screen
- Food spawns randomly
- Game over if the snake hits walls or itself

## Technologies

- HTML
- CSS
- JavaScript (DOM manipulation & event handling)

## How to Play

1. Clone or download the repository.
2. Open `index.html` in your browser.
3. Use **arrow keys** to move the snake.
4. Eat food to increase your score. Avoid collisions with walls or your own tail.

## Optional Enhancements

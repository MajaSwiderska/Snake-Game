# 🐍 Pink Snake Game

A colorful and playful Snake Game built using HTML, CSS, and JavaScript. This version features an **extra cute** cartoon-style light purple snake with sparkling eyes, rosy cheeks, and a happy smile! Eat raspberries, grow your snake, and enjoy the grass-themed game board. The game works on desktop and mobile devices, with both keyboard and touch controls.

## ✨ Current Features

-   **🐍 Cartoon Snake** – A light purple snake with big sparkly eyes, a big smile, rosy cheeks, and even a tiny floating heart above its head! Each body segment has a soft gradient and inner shine.
-   **🍓 Raspberry Food** – Eat delicious raspberries (with a little sparkle) to increase your score and grow longer.
-   **🌿 Grass Background** – The game board features a grassy outdoor theme for a fresh, playful feel.
-   **📱 Mobile Controls** – Play using the large, pastel-colored on-screen arrow buttons on mobile devices. They have a satisfying "press" animation.
-   **⌨️ Keyboard Support** – Use the arrow keys to control the snake on desktop.
-   **📊 Score Counter** – The score is displayed at the top (though the HTML for it was in a previous version, the game logic tracks it).
-   **💖 Adorable Aesthetics** – Includes floating hearts in the background, soft shadows, and a cute "Game Over!" overlay with a restart button.

## 🕹️ How to Play

1.  Open the game in your browser.
2.  Control the snake:
    -   **Desktop:** Use the **arrow keys** (⬆️⬇️⬅️➡️).
    -   **Mobile:** Tap the **on-screen arrow buttons**.
3.  **Eat the raspberries** 🍓 to grow longer and increase your score.
4.  **Avoid:**
    -   Hitting the **walls**
    -   Running into your own **snake's body**
5.  If you crash, the game ends and you can click "Play Again ✨" to restart.

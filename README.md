# 🐍 Pink Snake Game

A colorful and playful **Snake Game** built using **HTML, CSS, and JavaScript**.
This version features a **cartoon-style light purple snake**, **raspberry food**, and a **grass-themed game board**. The game works on **desktop and mobile devices**, with both **keyboard and touch controls**.

---

## 🎮 Features

* 🐍 **Cartoon Snake** – A light purple snake with cute eyes.
* 🍓 **Raspberry Food** – Eat raspberries to increase your score.
* 🌿 **Grass Background** – The game board has a grassy look for a fun outdoor theme.
* 📱 **Mobile Controls** – Play using on-screen arrow buttons on mobile devices.
* ⌨️ **Keyboard Support** – Use arrow keys to control the snake on desktop.
* 📊 **Score Counter** – Track how many raspberries you collect.

---

## 🕹️ How to Play

1. Open the game in your browser.
2. Control the snake:

   * **Desktop:** Use the **arrow keys**.
   * **Mobile:** Use the **on-screen arrow buttons**.
3. Eat the **raspberries** to grow longer and increase your score.
4. Avoid:

   * Hitting the **walls**
   * Running into **your own snake**

If you crash, the game ends and your final score appears.
